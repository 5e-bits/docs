Getting Started

Let's make our first API request to the D&D 5th Edition API!

Open up a terminal and use cURL, PowerShell, or httpie to make an API request for a resource. You can also scroll through the definitions below and send requests directly from the endpoint documentation!

For example, if you paste and run this terminal command:

  • cURL
  • PowerShell
curl -X GET "https://www.dnd5eapi.co/api/2014/ability-scores/cha" -H "Accept: application/json"

# or with httpie
http dnd5eapi.co/api/2014/ability-scores/cha
Invoke-RestMethod -Uri https://www.dnd5eapi.co/api/2014/ability-scores/cha
